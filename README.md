이미지 뷰어입니다, 프롬프트 확인을 곁들인

폴더의 이미지들을 5행 4열 형태로 확인할 수 있고, 이미지를 클릭하면 모달형태로 전체 이미지와 프롬프트를 확인할 수 있습니다.
이미지가 많아지면 성능 이슈가 있기 때문에, 최신 이미지만 확인하고 싶다면 마지막 100장 불러오기를 추천드립니다.
이미지 셔플은 폴더 내의 이미지의 순서를 섞어서 확인할 수 있는 기능입니다.
